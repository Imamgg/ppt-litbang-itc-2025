# JavaScript

# **Apa itu JavaScript?**

JavaScript adalah bahasa pemrograman yang digunakan developer untuk membuat halaman web yang interaktif. Dari menyegarkan umpan media sosial hingga menampilkan animasi dan peta interaktif, fungsi JavaScript dapat meningkatkan pengalaman pengguna situs web. Sebagai bahasa skrip sisi klien, JavaScript adalah salah satu teknologi inti dari World Wide Web. Misalnya, saat menjelajah internet, kapan pun Anda melihat carousel gambar, menu tarik-turun klik untuk menampilkan, atau warna elemen yang berubah secara dinamis di halaman web, Anda melihat efek JavaScript.

# Program Hello World

```jsx
console.log("Hello World"); // Menampilkan Hello World di console
document.write("Hello World"); // Menampilkan Hello World di halaman web
```

Dari contoh diatas javascript dapat menampilkan Hello World di dua tempat yang berbeda console.log() akan menampilkan di console, sedangkan document.write() akan menampilkan ke halaman web.

# Variabel dan Tipe data

---

Variabel sendiri merupakan tempat untuk menyimpan suatu value/data, sedangkan tipe data sendiri terdiri dari: 

- String
- Number
- Boolean
- Array
- Object
- Null,
- Undifined

## Variabel

Cara mendeklarasikan variabel sendiri ada dua cara yaitu menggunakan syntax let untuk variabel yang dapat diubah valuenya dan const untuk variabel constant.

## String | ‘ ‘ | “ ”

String merupakan tipe data yang mewakilkan kata/kalimat, untuk mendeklarasikan tipe data string kita bisa menggunakan tanda petik satu ‘ ’ ataupun petik dua “ ” value dari tipe data string bisa kita masukkan di antara petik satu ataupun petik dua.

```jsx
let name = "your name"; // variabel let yang dapat diubah valuenya
const name = "your name"; // variabel const yang constant

name = "jonh doe"; // menambahkan string ke variabel
```

## Number | Int | Float

Number terbagi menjadi dua yaitu integer dan float. Integer adalah angka/bilangan bulat, sedangkan float adalah angka/bilangan pecahan.

```jsx
let age = 19; // integer
const PI = 3.14; // float
```

## Boolean | true | false

Boolean adalah tipe data yang memiliki dua nilai yaitu true dan false. true merepresentasikan nilai benar, sedangkan false merepresentasikan nilai salah.

```jsx
const isJavaScript = true;
const isPhp = false;
```

## Array | [ ]

Array adalah tipe data yang menampung tipe data lain bahkan kita bisa menambahkan tipe data array lagi, adapun untuk membuat array kita bisa menggunakan kurung siku [ ], index array di javascript di mulai dari 0.

```jsx
const data = ["your name", 19, true, [1, 2, 3]];

// cara mengakses value array
console.log(data[0]); // akan menampilkan value index ke-0 yaitu "your name"
console.log(data[1]); // 19
console.log(data[2]); // true
console.log(data[3][0]) // nested array 1
```

## Object | { }

Object adalah tipe data yang memiliki kesamaan dengan array yaitu tipe data yang dapat menampung tipe data lain bahkan tipe data object itu sendiri, adapun untuk membuat object kita bisa menggunakan kurung kurawal { }, perbedaan array dengan object yaitu array menggunakan index untuk mengakses suatu value didalam array, sedangkan object menggunakan key untuk mengakses sebuah value, jadi setiap data yang ada didalam object memiliki key yang merupakan kunci untuk mendapat value yang ada di object tersebut.

```jsx
const obj = {
		name: "your name",
		age: 19,
		married: false,
		skills: {
				frontEnd: ["html", "css", "javascript"],
				backEnd: ["python"],
		},
}

console.log(obj.name); // your name
console.log(obj.age); // 19
console.log(obj.married); // false
console.log(obj.skills); // {frontEnd: ["html", "css", "javascript"], backEnd: ["python"],}
console.log(obj.skills.fontEnd); // ["html", "css", "javascript"]
```

## Null | kosong

Null merupakan tipe data yang tidak memiliki nilai alias kosong, berbeda dengan nol, nol sendiri memiliki nilai yaitu 0.

```jsx
const kosong = null;
const nol = 0;
```

## Undefined | tidak didefinisikan

Undefined adalah tipe data yang tidak terdefinisi, undefined berbeda dengan null dimana undefined memiliki nilai tetapi nilainya tidak terdefinisikan.

```jsx
const defined = "defined";
const notDefined = undefined;
```

# Operator Perbandingan

---

Operator perbandingan digunakan untuk membandingkan 2 buah nilai atau lebih yang akan menghasikan nilai boolean. adapun operator perbandingan sebagai berikut:

- <              Lebih kecil dari
- >              Lebih besar dari
- ≤              Lebih kecil sama dengan
- ≥              Lebih besar sama dengan
- ==           Sama dengan (dalam kontexts nilai tanpa melibatkan tipe data)
- ≠              Tidak sama dengan (dalam kontexts nilai tanpa melibatkan tipe data)
- ===        Sama dengan (dalam kontexts nilai dan tipe data)
- !==          Sama dengan (dalam kontexts nilai dan tipe data)

```jsx
console.log(1 < 2); // true
console.log(1 > 2); // false
console.log(1 <= 2); // true
console.log(1 >= 2); // false
console.log(1 == "1"); // true
console.log(1 != "1"); // false
console.log(1 === "1"); // false
console.log(1 !== "1"); // true
```

# Operator Logika/Gerbang Logika

---

Operator Logika akan menghasikan nilai boolean samahalnya dengan operator perbandingan, tetapi ada sedikit perbedaan antara lain:

- &&          Kedua nilai yang dibandingkan harus bernilai true agar mendapatkan hasil true
- |  |            Salah satu nilai yang dibandingkan harus bernilai true agar mendapatkan hasil true
- !               Mengubah nilai true menjadi false begitupun sebaliknya

```jsx
console.log(true && true); // true
console.log(true || false); // true
console.log(!true); // false
```

# Perkondisian/Percabangan

---

perkondisian adalah block code yang akan dieksekusi/dijalankan ketika kondisi bernilai true, perkondisian sendiri terbagi menjadi 2 yaitu:

- IF ~ Else if ~ Else
- Switch

## IF ~ Else If ~ Else

```jsx
const name = "jonh doe";

if (name === "jonh doe") {
	console.log("Hello jonh doe");
} else if (name === "admin") {
	console.log("Hello admin");
} else {
	console.log("Hello World");
}
```

## Switch

```jsx
const nilai = 90;

switch (nilai) {
	case 90:
		console.log("A");
		break;
	case 80:
		console.log("B");
		break;
	/* more conditon */
	default:
		console.log("NaN");
		break;
};
```

# Perulangan/Pengulangan

---

Perulangan adalah blok code yang akan diulang sebanyak yang kita tentukan atau yang tidak kita tentukan, perulangan terbagi menjadi 3 yaitu:

- For
- While
- Do ~ While

## For

```jsx
for(let i = 1; i <= 10; i++) {
	console.log("Hello World  - " + i); 
};
```

## While

```jsx
let i = 1;
while(i <= 10) {
	console.log("Hello World  - " + i);
	i++;
};
```

## Do ~ While

```jsx
let i = 1;
do {
	console.log("Hello World  - " + i);
	i++;
} while(i <= 10);
```

# Function

---

Function adalah blok code yang dibaut/dirancang untuk mengerjarkan suatu program yang dapat dipanggil ketika kita membutuhkannya. adapun cara membuat function di javascript ada 3 cara yaitu:

- Function Expressions
- Function Hoisting
- Arrow Function (ES6)

```jsx
// Function Expressions
const funcExpressions = function () {
	console.log("this is function expressions");
};

// Function Hoisting
function funcHoisting() {
	console.log("this is function hoisting");
};

// Arrow Function (ES6)
const ArrowFunc = () => {
	console.log("this is arrow function");
};

// called function
funcExpressions();
funcHoisting();
ArrowFunc();
```

## Parameter

Parameter adalah data yang di tangkap oleh function, value dari parameter bisa kita gunakan di dalam sebuah function asalkan parameternya dikirim pada saat pemanggilan function.

```jsx
function sayHello(name) {
	console.log("Hello " + name); // Hello your name
};

sayHello("your name");
```

Parameter juga bisa kita berikan nilai default jika pada saat pemanggilan function kita tidak mengirimkan argument apapun maka nilai default lah yang akan digunakan di dalam sebuah function.

```jsx
function sayHello(name = "admin") {
	console.log("Hello " + name); // Hello admin
};

sayHello();
```

Function juga dapat mengembalikan sebuah nilai dengan menggunakan syntax return untuk mengembalikan suatu nilai yang kemungkinan nilai tersebut akan dipakai di function yang lain.

```jsx
function getName() {
	return "your name";
};

function sayHello(name = "admin") {
	console.log("Hello " + name); // Hello your name
};

const name = getName();
sayHello(name);
```

Function juga memiliki parameter arguments yang akan menangkap semua parameter yang dikirim ketika pemanggilan function tetapi tidak ditangkap oleh function tersebut.

```jsx
function concatNumber() {
	let result = 0;
	for(let i = 0; arguments.lenght; i++) {
		result += arguments[i];
	};
	
	return result;
};

console.log(concatNumber(1, 2, 3, 4, 5)); // 15
```
